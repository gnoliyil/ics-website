How to read paper?

1. Knuth - Mathematical Analysis of Algorithms (P1-P18)

主讲: 李皓琳 王海滨 郭秀甲 3/11 3/18 讲稿1 讲稿2

导读问题(加*部分选做)

  1. 什么是in-situ permutation？它的要求是什么？
  2. 对于文章给出的算法，
    1. 证明它的正确性.
    2. a, b 的极值在什么情况下取到？
    3. a, b 的均值、* 方差怎样求出？
  3. 文章给出了哪些优化/情形？这些情形下算法的复杂度怎样求？
  4. * 如果空间是 O(log n) ，有什么更优的方法？
  5. * 文章证明部分有一处（或者几处）细节错误，你能找到吗？

2. Dean and Ghemawat - MapReduce: Simplified Data Processing on Large Clusters

主讲: 沈程 王子辰 3/25 讲稿下载(新版)

  1. MapReduce算法的框架
  2. MapReduce在集群上运行时考虑了哪些因素？例如怎样实现容错，效果如何等等。
  3. 设计MapReduce算法解决问题:
    • Sorting, Searching, Indexing, Classification, TF-IDF
    • 社交网络：输入（a→b，a关注b），输出所有的（a↔b）
    • 前缀和：输入（i , ai），每台机器内存大小O(nc)
    • 矩阵乘法

3. Fast Fourier Transform

主讲: 杨旭 曹宽 4/1 讲稿

  1. 了解什么是傅里叶变换，离散傅里叶变换
  2. 问题
    • 一个 N 维向量的 Fast Fourier Transform(FFT) 的计算复杂度是什么？
    • 对于 N = N1·N2 的情况，请推导出 FFT 的计算复杂度。
    • 对于 N = 2n 的情况，解释递归 FFT 算法的求解过程。
      • 对于 N = 2n 的情况，解释迭代 FFT 算法的求解过程。
